Steps to create API

1. Create a new project in Visual Studio

2. Create ASP.Net WEB Application (.NetFramework) Template select it

3. Click on Create Button

4. Choose Web Api Template Options

5. Click on Next Button

6. Web Application Project Created

7. Select Models Folder and Right Click on it.

8.Click on Add

9. Select Ado.Net Entity Data Model

10. Specify the name for item like EmployeeModel

11. Click Ok Button

12. Choose EF Designer from Database click on Next Button

13. Click on New Connection

14. Paste Server Name from Database

15. click on trust server certificate checkbox

16. Select Database Namelike EmployeeModel

17. Go ForNext

18.Save ConnectionString checkbox select it

19. Press Next button two times

20. select the table name

21. click on Finish button

22. Select Controllers Folder and Right Click on it.

22. Click on Controller Option

23. Add new Scaffold ITem will open

24. click on web api

25. select webapi controller with action,using entity framework

26.click on add button

27. choose model class and context class name

28. click on add

29. your controller with crud operation method will be created
